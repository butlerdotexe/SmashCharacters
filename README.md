Smash Bros Quiz

Purpose: Take a quiz to find out what character fits your playstyle the most.

Techonologies Used:
-AJAX
-Node.js
-Express.js
-HTML
-CSS
-JavaScript
-MySQL
-jQuery

How To Run: Run index.js on either node or nodemon. Go to localhost:8000 to access the local server.
