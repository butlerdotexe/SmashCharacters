How it Works

We first ask the user for the different playstyles that they are interested in. (All-Rounders, Rushdown, Hit & Run, Etc...)

The following questions will ask for different specifing characteristics, returning it as a list of corresponding numbers. From there, we iterate over each character, tallying every time a character matches a characteristic. At the end of the iteration, we sort each character from largest to smallest number of matches.

Your results are based off of the best matches.
